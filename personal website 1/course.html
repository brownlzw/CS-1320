CS 1270 Database Management System

The amount of information that must be processed in many applications is exploding. A modern database managementn system (DBMS) provides the scalability that is required by these applications. As a result, DBMS’s are now ubiquitous in modern industrial practice. This course examines the practice of database management through two major units. The first discusses the users view (externals) of a DBMS. It covers concepts that are fundamental to the proper use of a DBMS including database languages and database design. The second discusses what goes on inside (internals) a DBMS. It covers algorithmic detail for some of the key components of a DBMS.

ENGN 2912B Scientific Programming in C++

The primary aim of this course is to gain practical experience with programming in the C++ language. In particular, students will solve numerical and scientific computing problems using modern C++ and commonly used scientific computing and visualization libraries. Students will become proficient with the use of common programming toolsets for compiling, linking, debugging, and managing source code in a collaborative software development environment. Final projects may utilize Brown University’s Center for Computation and Visualization (CCV) to explore more advanced scientific computing topics, such as parallel computing with multi-threaded applications, machine learning with convolutional neural networks (CNN), 3D visualization of large data sets, or algorithm acceleration with graphical processing units (GPU).

CS 1320 Creating Modern Web Applications

This course takes a holistic look at the process of developing web and mobile applications and aims to bring the students to a point of mastery of many of the most used web technologies, and development practices. The course includes a semester long group final project in which the students will be working with external companies, non-profits, and other organizations.

Machine Learning

This course covers fundamental topics in pattern recognition and machine learning. Topics include: decision theory, parametric and non-parametric learning, dimensionality reduction, graphical models, exact and approximate inference, semi-supervised learning, generalization bounds and support vector machines. We will consider applications in computer vision, signal processing, speech recognition and information retrieval.
